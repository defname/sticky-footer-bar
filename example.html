Sticky Footer Bar

About

Sticky Footer Bar is a little jQuery-plugin to make a footer "sticky". This means a part of the footer is always displayed on the bottom of the screen and if you scroll down under its actual position, it's scrolling too like the rest of the document.

Look at this page itself to understand what I mean ;-)

Hints

  • Set the CSS margin-property for all elements within your footer (and for the footer itself) to zero and use padding instead. Since margin are collapsing the calculation of the height will cause problems.
  • sticky-footer-bar takes two (optional) parameters. The first is either a number telling the script how much of the footer should be displayed on the bottom of the screen or a jQuery-selector-string or a jQuery object. In this case the height of the specified object is used.
  • Whenever the footer is in "sticky-mode" (what I mean is when the footer isn't displayed the usual way) the script adds the CSS-class sticky to it. With the second parameter you can change the classname that should be used.
